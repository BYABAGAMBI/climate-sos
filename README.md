# Climate Africa SOS (U) Ltd Website

A professional, mobile-friendly website for Climate Africa SOS (U) Ltd, a non-profit environmental organization based in Kanungu District, Uganda.

## Project Overview

This website showcases Climate Africa SOS's mission to combat climate change and empower communities through environmental conservation, wetland restoration, tree planting, renewable energy promotion, and youth/women entrepreneurship training.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with green and blue color scheme
- **Fast Loading**: Optimized for performance even on slow internet connections
- **Accessibility**: Built with accessibility best practices
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## Pages

1. **Home** (`index.html`)
   - Organization vision and mission
   - Introduction to climate action work
   - Key statistics and focus areas
   - Call-to-action for involvement

2. **About Us** (`about.html`)
   - Organization background and history
   - Core objectives and values
   - Organizational structure
   - Leadership team information

3. **Our Programs** (`programs.html`)
   - Wetland restoration (fish ponds, beekeeping)
   - Tree planting & avocado seedling distribution
   - Fuel-efficient cookstoves
   - Youth & women entrepreneurship training

4. **Projects & Impact** (`impact.html`)
   - Key achievements and success metrics
   - Environmental conservation results
   - Success stories and testimonials
   - Partner organizations

5. **Get Involved** (`get-involved.html`)
   - Volunteer opportunities
   - Partnership options
   - Donation information
   - Donor recognition levels

6. **Contact** (`contact.html`)
   - Contact information and office location
   - Contact form
   - Directions and office hours
   - Multiple contact methods

## Technical Details

### Technologies Used
- **HTML5**: Semantic markup for accessibility and SEO
- **CSS3**: Modern styling with gradients, animations, and responsive design
- **JavaScript**: Interactive features and form validation
- **Google Fonts**: Inter font family for professional typography

### Design Features
- **Color Scheme**: Green (#2c7a2c, #4ade80) and blue (#1a5490, #60a5fa) representing environment and sustainability
- **Typography**: Clean, readable fonts with proper hierarchy
- **Layout**: Grid and flexbox for responsive layouts
- **Animations**: Subtle hover effects and scroll animations
- **Forms**: Validated contact and donation forms

### Responsive Breakpoints
- **Desktop**: 1024px and above
- **Tablet**: 768px - 1023px
- **Mobile**: Below 768px

## Getting Started

1. **Download the Files**: All website files are contained in this directory
2. **Local Development**: Open `index.html` in a web browser to view the site
3. **Web Server**: For production, deploy files to a web server

## Customization

### Colors
Main colors are defined in `styles.css`:
- Primary Green: `#2c7a2c`
- Light Green: `#4ade80`
- Primary Blue: `#1a5490`
- Light Blue: `#60a5fa`

### Content
- Text content can be edited directly in HTML files
- Images should be placed in appropriate directories
- Contact information should be updated with real details

### Forms
- Contact forms are client-side validated
- Backend integration needed for actual email sending
- Donation forms require payment gateway integration
